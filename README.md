# 💼 Vue Jobs

A job listing web app built with **Vue 3, Vue Router, Tailwind CSS, and JSON Server** as a mock REST API.

---

## 🔥 About the Project

Vue Jobs is a CRUD job board I built while learning Vue 3. It covers the full cycle of creating, reading, updating, and deleting job listings, all wired up to a mock JSON API so it behaves like a real application.

---

## ✨ Features

- Browse all job listings
- View full job details
- Add new job listings
- Edit existing listings
- Delete listings
- Mock REST API with JSON Server (no backend needed)
- Fully responsive with Tailwind CSS
- Client-side routing with Vue Router

---

## 🛠 Tech Stack

| Technology | Purpose |
|---|---|
| Vue 3 | Frontend framework |
| Vue Router | Client-side routing |
| Tailwind CSS | Styling |
| JSON Server | Mock REST API |
| Vite | Build tool |

---

## ⚙️ Getting Started

```bash
# Clone the repository
git clone https://github.com/kamran-faiz/vue-jobs-project.git
cd vue-jobs-project

# Install dependencies
npm install

# Start JSON Server (mock API) on port 5000
npx json-server --watch db.json --port 5000

# In a separate terminal, start the Vue dev server
npm run dev
```
